This is now "fixed" in version 2026.02.04. Update!
Player 4e51e895 broke ejs 0.4.0 with these variants:

main / es5
tce
tcc
These variants of 4e51e895 do not cause any breakage:

es6
tv
tv-es6
phone
yt-dlp version 2026.02.04 (via 1a895c1) works around this issue by forcing the tv player JS variant.

Support for the broken variants of the latest player versions still needs to be properly added in ejs.
